# AI Hypotheses

This directory contains AI-generated theoretical proposals and research hypotheses that have been developed beyond their initial issue submission.

## Contents

Hypotheses are organized by topic and development status. Each file should include:
- Clear hypothesis statement
- Theoretical foundation
- Supporting analysis
- Testable predictions
- AI attribution and context

## Workflow

1. **Origin**: Hypothesis submitted via [AI Hypothesis issue template](../../.github/ISSUE_TEMPLATE/ai-hypothesis.md)
2. **Development**: If hypothesis warrants expansion, detailed version created here
3. **Cross-reference**: Maintain links between issue and developed version
4. **Status tracking**: Use file naming to indicate development status

## Naming Convention

- `[topic]-[brief-description].md` for main content
- `[topic]-[brief-description]-discussion.md` for ongoing dialogue
- `[topic]-[brief-description]-validation.md` for testing/verification attempts
