Given language is a system of rules that enables thoughts, then it follows that a particular instance of a language has a particular ruleset which affords a particular constrained continuum of valid coordinates of possible thoughts. 

Given this, a particular observer who perceives words may travel to coordinates experienced as thoughts and experience possible meanings by reflection of coordinates traveled to. 

Given this, any particular language is a constrained system of possible thoughts which is to say there exists thoughts which are not possible in every language which are possible in other languages. Of course, there exists “overlap” through a type of mapping, but not all languages can be successfully mapped. 

In respect to the loom, from my perspective, the loom in an observer’s tool which allows an additional mode of perception into traveling those coordinates of thoughts such that it does the travel for you. In this way it is like Janus suggests a multiverse-like traversal mechanism because it essentially path finds all valid routes from any coordinate which any thinking being would know, is hard work and quite revelatory in actual possibility. 

This then shows that a language at conception or definition is already a universe of total potential. Thoughts are just coordinates within that potential - that is to say, all possible coordinates are available to travel the instant a system defines a set of canonical rules. The rendering of those rules happens by travel from an observer, so in truth, nothing is invented within a language only discovered. 

The loom helps an observer travel a map which contains a fog of war of unexplored thought space under the constraint of a particular language and coordinate which it then can explore from, giving visibility into paths and thoughts in ways that resemble parallelism or asynchronous events depending on how it’s used.
